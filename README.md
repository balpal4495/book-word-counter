# Book word counter

## Stack
* node v6.0.0
* yarn v0.21.3
* npm v3.8.6

# Run instructions
install dependencies

```
npm install 
```
or 

```
yarn
```

```
node attempt1.js
```
or other corresponding filenames, each file name corresponds to test[n].json
so attempt1.js -> test1.json

## Test

  ```
  yarn test
  ```
or 
  ```
  npm test
  ```
